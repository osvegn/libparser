# libparser
Simple libc parser to parse args from command line.

## Platform
Linux only for the moment. It will be improved.

## How to build
### Build lib
```bash
make
```

### Build and run tests
```bash
make run_tests
```

## Contribute
No specification, ask for feature by creating issue or creating pull requests with changes.
